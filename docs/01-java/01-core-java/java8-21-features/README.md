# Java Features Roadmap (8 to 21)

Explore the evolution of Java through its most impactful version releases. These guides follow an active learning pattern to help you retain modern Java history.

## Major Release Ranges

### 1. [Java 8: The Functional Leap](./java8.md)
- Lambda Expressions
- Streams API
- Optional
- Date & Time API

### 2. [Java 9-11: The Modern Foundation](./java9-11.md)
- Modularity (Project Jigsaw)
- Local Variable Type Inference (`var`)
- New HTTP Client
- 6-month release cycle

### 3. [Java 12-17: The Productivity Era](./java12-17.md)
- Records (Boilerplate reduction)
- Sealed Classes (Architecture control)
- Text Blocks
- Switch Expressions

### 4. [Java 18-21: The Concurrency Era](./java18-21.md)
- Virtual Threads (Project Loom)
- Record Patterns
- Sequenced Collections
- Performance optimizations
