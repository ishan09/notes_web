# TypeScript

TypeScript adds static typing to JavaScript. This module covers the type system, generics, utility types, and advanced patterns used in real projects.

## Module Index

2. [Generics](./02-generics.md)

4. [Advanced Types & Type Guards](./04-advanced-types.md)

## What You Should Know After This Module

- Distinguish `interface` vs `type` alias and know when to use each
- Write generic functions and classes
- Use built-in utility types (`Partial`, `Required`, `Pick`, `Omit`, `Record`, etc.)
- Narrow types with type guards, discriminated unions, and assertion functions
- Configure `tsconfig.json` for strict mode
