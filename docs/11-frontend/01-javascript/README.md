- Explain and use closures, prototypal inheritance, and the event loop
- Write and reason about async code using Promises and async/await
- Know the differences between `var`, `let`, and `const`
- Understand `this` binding in different contexts
- Use modern ES6+ features (destructuring, spread, optional chaining, nullish coalescing)
