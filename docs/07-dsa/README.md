# DSA Patterns - Complete List

This folder contains comprehensive guides for 16 essential Data Structures & Algorithms patterns that cover 90%+ of coding interview problems.

## Pattern List

### Array/String Patterns
1. **[Sliding Window](./01-sliding-window.md)** - Contiguous subarray/substring problems
2. **[Two Pointers](./02-two-pointers.md)** - Pair finding, array manipulation
3. **[Fast & Slow Pointers](./03-fast-slow-pointers.md)** - Cycle detection, middle finding

### Interval & Sorting Patterns
4. **[Merge Intervals](./04-merge-intervals.md)** - Overlapping intervals, scheduling
5. **[Cyclic Sort](./05-cyclic-sort.md)** - Finding missing/duplicate numbers

### LinkedList Patterns
6. **[In-place Reversal of LinkedList](./06-linkedlist-reversal.md)** - Reverse operations

### Tree Patterns
7. **[Tree Breadth First Search (BFS)](./07-tree-bfs.md)** - Level-order traversal
8. **[Tree Depth First Search (DFS)](./08-tree-dfs.md)** - Pre/In/Post-order traversal

### Heap Patterns
9. **[Two Heaps](./09-two-heaps.md)** - Median finding, scheduling

### Combinatorial Patterns
10. **[Subsets](./10-subsets.md)** - Combinations, permutations, power set

### Search Patterns
11. **[Modified Binary Search](./11-modified-binary-search.md)** - Rotated arrays, range search
12. **[Top 'K' Elements](./13-top-k-elements.md)** - K largest/smallest elements

### Bit Manipulation
12. **[Bitwise XOR](./12-bitwise-xor.md)** - Single number, missing number

### Merge Patterns
14. **[K-way Merge](./14-k-way-merge.md)** - Merge K sorted arrays/lists

### Dynamic Programming
15. **[0/1 Knapsack](./15-knapsack-dp.md)** - Subset sum, partition problems

### Graph Patterns
16. **[Topological Sort](./16-topological-sort.md)** - Dependency resolution, course schedule

## How to Use These Patterns

1. **Learn the pattern** - Understand the core concept and when to apply it
2. **Master the template** - Each pattern has a code template
3. **Practice problems** - Start with beginner, progress to advanced
4. **Mix patterns** - Real interviews often combine multiple patterns

## Quick Reference

| Pattern | Time Complexity | Space Complexity | Key Use Case |
|---------|----------------|------------------|--------------|
| Sliding Window | O(n) | O(1) or O(k) | Contiguous subarrays |
| Two Pointers | O(n) | O(1) | Sorted arrays, pairs |
| Fast & Slow | O(n) | O(1) | Cycle detection |
| Merge Intervals | O(n log n) | O(n) | Overlapping intervals |
| Cyclic Sort | O(n) | O(1) | Missing numbers |
| LinkedList Reversal | O(n) | O(1) | Reverse operations |
| Tree BFS | O(n) | O(n) | Level-order |
| Tree DFS | O(n) | O(h) | Pre/In/Post-order |
| Two Heaps | O(n log n) | O(n) | Median, scheduling |
| Subsets | O(2^n) | O(2^n) | Combinations |
| Modified Binary Search | O(log n) | O(1) | Rotated/modified arrays |
| Bitwise XOR | O(n) | O(1) | Single number |
| Top K Elements | O(n log k) | O(k) | K largest/smallest |
| K-way Merge | O(n log k) | O(k) | Merge K lists |
| 0/1 Knapsack | O(n*W) | O(n*W) | Subset sum |
| Topological Sort | O(V+E) | O(V+E) | Dependencies |
