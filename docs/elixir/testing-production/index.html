Testing and Production

This module focuses on how to validate, deploy, and operate Elixir applications safely.

What You Should Know After This Module​

  • Write practical tests with ExUnit
  • Use property-based testing where it adds value
  • Build and run releases
  • Observe behavior with logs, metrics, and traces
  • Reason through production failure scenarios and system design trade-offs
  • Diagnose live production issues: mailbox bloat, memory leaks, hot processes, GenServer state
