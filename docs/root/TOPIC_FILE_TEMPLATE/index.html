Embedded Learning Techniques​

This template incorporates:

  1. Active Recall

    • "Self-Check Questions" - answer from memory
    • "Try it" prompts after code examples
    • "Stop and think" pauses

  2. Feynman Technique

    • Simple explanation first
    • Real-world analogies
    • "Explain in your own words" prompts

  3. Elaboration

    • "Why This Matters" section
    • Trade-offs table
    • "Compare" prompts
    • Connection to other topics

  4. Spaced Repetition

    • "Before you start" prerequisite check
    • "How This Connects" section
    • Links to related topics for review

  5. Immediate Practice

    • "Try it" exercises after each concept
    • Progressive practice levels
    • Scenario-based questions

  6. Interleaving

    • "Related concepts" comparisons
    • "Compare" prompts
    • Links encouraging topic mixing

Writing Guidelines​

DO:

  • Start with simple language, build to technical
  • Use concrete examples and analogies
  • Ask questions throughout ("Can you...", "What if...", "Why...")
  • Link to related content
  • Include visual diagrams where helpful
  • Show trade-offs explicitly
  • Provide immediate practice opportunities

DON'T:

  • Start with jargon or formal definitions
  • Just list facts without context
  • Put all questions at the end
  • Provide answers immediately after questions
  • Skip the "why it matters" context
  • Forget to show when NOT to use something
