DSA Patterns - Complete List

This folder contains comprehensive guides for 16 essential Data Structures & Algorithms patterns that cover 90%+ of coding interview problems.

Pattern List​

Array/String Patterns​

  1. Sliding Window - Contiguous subarray/substring problems
  2. Two Pointers - Pair finding, array manipulation
  3. Fast & Slow Pointers - Cycle detection, middle finding

Interval & Sorting Patterns​

  1. Merge Intervals - Overlapping intervals, scheduling
  2. Cyclic Sort - Finding missing/duplicate numbers

LinkedList Patterns​

  1. In-place Reversal of LinkedList - Reverse operations

Tree Patterns​

  1. Tree Breadth First Search (BFS) - Level-order traversal
  2. Tree Depth First Search (DFS) - Pre/In/Post-order traversal

Heap Patterns​

  1. Two Heaps - Median finding, scheduling

Combinatorial Patterns​

  1. Subsets - Combinations, permutations, power set

Search Patterns​

  1. Modified Binary Search - Rotated arrays, range search
  2. Top 'K' Elements - K largest/smallest elements

Bit Manipulation​

  1. Bitwise XOR - Single number, missing number

Merge Patterns​

  1. K-way Merge - Merge K sorted arrays/lists

Dynamic Programming​

  1. 0/1 Knapsack - Subset sum, partition problems

Graph Patterns​

  1. Topological Sort - Dependency resolution, course schedule

How to Use These Patterns​

  1. Learn the pattern - Understand the core concept and when to apply it
  2. Master the template - Each pattern has a code template
  3. Practice problems - Start with beginner, progress to advanced
  4. Mix patterns - Real interviews often combine multiple patterns

Pattern Coverage​

These 16 patterns cover:

  • Arrays & Strings: Patterns 1, 2, 5, 11, 12
  • LinkedLists: Patterns 3, 6
  • Trees: Patterns 7, 8
  • Heaps: Patterns 9, 13
  • Graphs: Pattern 16
  • Dynamic Programming: Pattern 15
  • Combinatorics: Pattern 10
  • Intervals: Pattern 4
  • Merging: Pattern 14

Study Recommendation​

Week 1: Patterns 1-4 (Arrays, Pointers, Intervals) Week 2: Patterns 5-8 (Sorting, LinkedList, Trees) Week 3: Patterns 9-12 (Heaps, Subsets, Search, XOR) Week 4: Patterns 13-16 (Top K, Merge, DP, Graphs)
